# Data Dictionary

## routes

GO route metadata from GTFS `routes.txt`.

## stops

Stop and station metadata from GTFS `stops.txt`.

## trips

Trip metadata from GTFS `trips.txt`.

## scheduled_stop_times

Scheduled arrivals and departures from GTFS `stop_times.txt`.

## realtime_observations

GTFS Realtime trip update delay observations by trip, route, stop, and observed time.

## service_alerts

GTFS Realtime service alerts stored separately from delay observations because explicit delay reasons are not always available.

## weather_observations

OpenWeather observations for major GO service areas.

## prediction_logs

Prediction requests and model outputs for monitoring and future evaluation.
