# Product Plan

## MVP Goal

Let a GO Transit commuter choose a route, optional stop, and travel time, then see delay probability, expected delay, confidence, reliability score, active alerts, and historical delay charts.

## Build Order

1. FastAPI backend
2. Database models
3. GTFS static import
4. Routes and stops endpoints
5. Frontend selectors
6. Realtime collector
7. Delay observations storage
8. Delay summary endpoint
9. Dashboard charts
10. Baseline predictor
11. Weather collection
12. ML training
13. ML-backed predictions when enough data exists
14. UI polish
15. README
